# qstui 
## 新分支：使用vite构建vue2项目
使用[underfin's vite-plugin-vue2](https://github.com/underfin/vite-plugin-vue2).

## Scripts

```bash
  # install
  yarn install

  # start dev server
  yarn dev
  # build for production
  yarn build # build for production
  # locally preview production build 本地预览production版本
  yarn serve

  # 测试
  yarn test

  # lint 带自动的fix和format
  yarn lint
```
